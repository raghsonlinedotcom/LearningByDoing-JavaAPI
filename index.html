java.lang.String

Immutable UTF-16 character sequence; common text utilities.

int length()

  • Purpose: Count the number of char values.
  • Parameters: none
  • Returns: length of this string.
  • Throws: —
  • Javadoc (summary): Returns the string’s character count as an int.
public class Demo {
  public static void main(String[] args) {
    String s = "Hello, Java!";
    System.out.println(s.length());
  }
}
13

String substring(int beginIndex, int endIndex)

  • Purpose: Extract a portion of the string.
  • Parameters: beginIndex inclusive, endIndex exclusive.
  • Returns: substring in range.
  • Throws: IndexOutOfBoundsException if indices are invalid.
  • Javadoc (summary): Returns a new string containing chars from the specified range.
public class Demo {
  public static void main(String[] args) {
    String s = "Transaction#INV-2025";
    System.out.println(s.substring(0, 12));  // "Transaction"
    System.out.println(s.substring(13));     // "INV-2025"
  }
}
Transaction INV-2025

  • Purpose: Test if the sequence occurs in this string.

  • Javadoc (summary): Checks for the presence of a CharSequence within the string.

}

java.util.List (ArrayList)

Ordered, index-based collection; allows duplicates.

boolean add(E e)

  • Purpose: Append an element.
  • Parameters: element to add.
  • Returns: typically true for lists that grow.
  • Throws: Optional UnsupportedOperationException for fixed-size lists.
  • Javadoc (summary): Adds the specified element to the end.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    List<String> list = new ArrayList<>();
    list.add("Apple");
    list.add("Orange");
    System.out.println(list); // [Apple, Orange]
  }
}
[Apple, Orange]

E get(int index)

  • Purpose: Access by position.
  • Parameters: zero-based index.
  • Returns: element at index.
  • Throws: IndexOutOfBoundsException for invalid index.
  • Javadoc (summary): Returns the element at the specified index.
import java.util.*;
public class Demo {
  public static void main(String[] args) {

    System.out.println(list.get(1)); // B
  }
}
B

E set(int index, E element)

  • Purpose: Replace the element at index.
  • Parameters: index; new element.
  • Returns: previously stored element.
  • Throws: IndexOutOfBoundsException.
  • Javadoc (summary): Assigns a new value at the given index.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    List<String> list = new ArrayList<>(List.of("A","B","C"));
    String old = list.set(1, "X");
    System.out.println(old + " -> " + list); // B -> [A, X, C]
  }
}
B -> [A, X, C]

E remove(int index) • boolean remove(Object o)

  • Purpose: Delete by position or by value.
  • Parameters: index or object to remove.
  • Returns: removed element / whether removal happened.
  • Throws: IndexOutOfBoundsException for index variant.
  • Javadoc (summary): Removes a single element by index or first match by value.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    List<String> list = new ArrayList<>(List.of("A","B","C","B"));
    list.remove(2);            // remove "C"
    list.remove("B");          // remove first "B"
    System.out.println(list);  // [A, B]
  }
}
[A, B]

default void sort(Comparator<? super E> c)

  • Purpose: In-place sort according to comparator.
  • Parameters: comparator (or null for natural order).
  • Returns: —
  • Throws: ClassCastException if elements not mutually comparable in natural order.

java.util.Map (HashMap)

Key-value mapping with fast average-time operations.

V put(K key, V value)

  • Purpose: Insert or replace value for key.
  • Parameters: key, value.
  • Returns: previous value or null.
  • Throws: NullPointerException for some map impls disallowing nulls.
  • Javadoc (summary): Associates value with key, returning old value if present.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    Map<String,Integer> m = new HashMap<>();
    System.out.println(m.put("apple", 2)); // null
    System.out.println(m.put("apple", 3)); // 2
    System.out.println(m);                 // {apple=3}
  }
}
null 2 {apple=3}

V computeIfAbsent(K key, Function<? super K,? extends V> f)

  • Purpose: Lazily create a value only if key is missing.
  • Parameters: key; mapping function.
  • Returns: existing or computed value.
  • Throws: if function throws; NullPointerException if null key when map forbids.
  • Javadoc (summary): If key absent, computes value with the function and stores it.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    Map<String,Integer> m = new HashMap<>();
    m.computeIfAbsent("apple", k -> 0);
    m.put("apple", m.get("apple") + 2);
    System.out.println(m); // {apple=2}
  }
}
{apple=2}

V merge(K key, V value, BiFunction<V,V,V> f)

  • Purpose: Combine existing and new values atomically.
  • Parameters: key, value to merge, merge function.
  • Returns: new value associated, or removed if function returns null.
  • Throws: if function throws.
  • Javadoc (summary): Merges value with existing mapping using a remapping function.
import java.util.*;
public class Demo {
  public static void main(String[] args) {
    Map<String,Integer> m = new HashMap<>(Map.of("banana",1));
    m.merge("banana", 3, Integer::sum);
    System.out.println(m.get("banana")); // 4
  }
}

java.util.stream.Stream

Functional operations: filter, map, distinct, sorted, collect.

<R> Stream<T> filter(Predicate<? super T> p)

  • Purpose: Keep elements satisfying predicate.
  • Parameters: predicate.
  • Returns: filtered stream.
  • Throws: if predicate throws.
  • Javadoc (summary): Returns a stream with elements that match the predicate.
import java.util.*;
import java.util.stream.*;
public class Demo {
  public static void main(String[] args) {
    List<String> words = List.of("zebra","apple","mango","kiwi");
    List<String> longWords = words.stream()
        .filter(w -> w.length() >= 5)
        .toList();
    System.out.println(longWords); // [zebra, apple, mango]
  }
}
[zebra, apple, mango]

<R> Stream<R> map(Function<? super T,? extends R> f)

  • Purpose: Transform elements.
  • Parameters: mapping function.
  • Returns: stream of mapped values.
  • Throws: if mapper throws.
  • Javadoc (summary): Applies a function to each element, yielding a stream of results.
import java.util.*;
import java.util.stream.*;
public class Demo {
  public static void main(String[] args) {
    List<String> words = List.of("zebra","apple","mango");
    List<Integer> lens = words.stream().map(String::length).toList();
    System.out.println(lens); // [5, 5, 5]
  }
}
[5, 5, 5]

<R,A> R collect(Collector<? super T,A,R> c)

  • Purpose: Terminal op to materialize results.
  • Parameters: collector (e.g., Collectors.toList()).
  • Returns: collected result.
  • Throws: if collector functions throw.
  • Javadoc (summary): Performs a mutable reduction operation on elements.
import java.util.*;
import java.util.stream.*;
public class Demo {
  public static void main(String[] args) {
    List<String> words = List.of("zebra","apple","mango");
    String joined = words.stream()
        .map(String::toUpperCase)
        .sorted()
        .collect(Collectors.joining(", "));
    System.out.println(joined); // APPLE, MANGO, ZEBRA
  }
}
APPLE, MANGO, ZEBRA
