Java Platform Module System (JPMS)

This tutorial walks you through the structure, setup, and execution of a real-world modular Java application using JPMS, introduced in Java 9.

Module Breakdown

Defines the domain object Book and exports the package com.lib.core.

module lib.core {
  exports com.lib.core;
}

Access layer for books. Depends on lib.core and exports com.lib.dao.

module lib.dao {
  requires lib.core;
  exports com.lib.dao;
}

Depends on both lib.core and lib.dao and encapsulates logic to fetch and print books.

module lib.service {
  requires lib.core;
  requires lib.dao;
  exports com.lib.service;
}

Depends only on lib.service and serves as the application entry point.

module lib.ui {
  requires lib.service;
}
